# Hi, I'm Abhinav Vummidichetty 👋

AI Engineer specializing in **Computer Vision**, **Generative AI**, and **High-Performance Systems**. I build robust, optimized architectures at the limit of compute.

---

- 🧠 **Agentic RAG Pipeline:** Multi-agent QA service using LangChain and a dense FAISS vector index of 10K+ documents with high-throughput FastAPI deployment.
- ⚡ **LLaMA-2 Edge Quantization:** Hardware-aware compression using LoRA/PEFT (90% param reduction) and 4-bit GPTQ for sub-200ms edge inference.
- 🔍 **PatchCore Anomaly Detection:** PyTorch-based unsupervised vision engine using WideResNet-50 feature maps, achieving 94.3% AUROC on MVTec benchmark.
- 🎯 **Pixel-Level Defect Segmentation:** Supervised U-Net localization model with a custom BCE + Dice loss function to resolve extreme class imbalances.
- 🎥 **Deterministic Video Perception:** CPU-optimized spatiotemporal computer vision pipeline using SSIM to segment 45-minute continuous feeds in under 2 minutes.
- 🧮 **Custom Mathematical OCR:** End-to-end CNN classification engine trained on HASYv2 (168 symbol classes) with advanced OpenCV structure localization.
- 🔄 **AutoEncoder Anomaly Scorer:** Unsupervised CNN AutoEncoder built to flag anomalies dynamically via nominal-state reconstruction MSE thresholding.
- 💾 **Customer Analytics Warehouse:** Multi-source enterprise warehouse on Snowflake built with dbt models and automated Python ETL schema validation.
- 📈 **Predictive Forecasting Pipeline:** Apache Airflow DAG orchestrating deep learning LSTM and ARIMA models for continuous operational forecasting.
- 📊 **Sales & Operations Dashboard:** Optimized Star Schema SQL data models paired with customized DAX-driven Power BI dashboards for real-time KPIs.
